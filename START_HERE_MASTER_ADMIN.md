## 🐛 TROUBLESHOOTING

### Can't see stats?
- Check internet connection
- Click refresh button (top right)
- Verify Firebase is connected

### Sidebar not showing?
- Click the expand/collapse arrow
- Check if you're on mobile (auto-collapses)

### Page not loading?
- Clear browser cache
- Check if logged in
- Try different browser
