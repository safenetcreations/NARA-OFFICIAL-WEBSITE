## 🎯 **Common Tasks**

### **1. Add a New Book**
```
1. Go to /admin/library/cataloguing
2. Click "Add New Book"
3. Fill Title + Material Type
4. Click "Save Book & Generate QR"
5. Click "Print QR Label"
6. Done! ✅
```

### **2. Search for a Book**
```
1. Go to /library
2. Enter search term
3. Or click a category
4. View results
5. Click for details
```

### **3. Check Book Availability**
```
1. Search for book
2. Click book title
3. See "Available Copies"
4. Green = Available
5. Red = Checked out
```

### **4. Print QR Labels**
```
1. Open book in cataloguing
2. See QR code on right
3. Click "Print QR Label"
4. Print dialog opens
5. Print and attach
```
